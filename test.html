Le développement web ?

Le développement web représente pour moi un univers sans limites, où toutes idées les plus farfelus peuvent prendre vie. C'est un monde sans cesse stimulant intellectuellement, qui change régulièrement, ce qui le rend constamment captivant. De mon année de terminale à ma deuxième année de BUT MMI, j'ai eu l'occasion de concrétiser de nombreux projets, mettant en pratique les théories enseignées ainsi que celles acquises en autonomie.

Dataviz

La 6ème extinction des espèces

Dans un contexte où l'activité humaine prend de plus en plus d’ampleur, de nombreuses espèces sont menacées d'extinction.

Objectif

Nous avons donc choisi de valoriser un jeu de données issus de la base de données du Muséum National d’Histoire naturelle. L’objectif est de rendre ces données accessibles au grand public à travers des graphiques interactifs.

Softskill :

Autonomie - Travail d’équipe - Gestion de projet - Autodidacte

Compétences développées

  • Réalisation d’une maquette de page web sur le logiciel Figma.
  • Manipulation d’un jeu de données avec des fichiers JSON
  • Réalisation de graphiques interactifs à partir de données en D3.js et en JS.
  • Réalisation d’une page mention légale

Référencement

Réflexion d'une meilleure SEO

Dans le web, l'importance de se démarquer des autres sites webs est primordiale. Dans lequel le référencement entre en jeu. Nous avons choisi d’analyser des sites dans un univers de décoration mignon.

Objectif

L’objectif est de réaliser une stratégie de référencement afin que les navigateurs le mettent en avant. Il était donc nécessaire de s'intéresser à la concurrence des sites communs à la nôtre, analyser les mots clés, les liens, trouver un nom de domaine adapté et un contenu permettant d’être valorisé et d’être mis, par exemple, dans des snippets.

Équipe de deux

Novembre 2023

BUT MMI2 - Semestre3

Langages & logiciels utilisés :

HTML - CSS - JS - D3.js - GSAP.js

Softskill :

Travail d’équipe - Analyse - Esprit critique

Compétences développées

  • Développer une stratégie de référencement
  • Utilisation de l’intelligence artificielle avec les meilleurs prompt à utiliser (autorisé par le professeur).
  • Être critique sur les meilleurs mots clés à choisir

Mon bilan

C’est un projet que je trouve très intéressant, surtout que le monde du référencement est un domaine où il faut faire de la veille très régulièrement, puisque les techniques d’aujourd’hui pourraient être obsolètes ou que les algorithmes de recherches peuvent être changés.
